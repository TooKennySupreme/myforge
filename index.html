A) Preparing - top

Make sure you have the following:

  1. PHP >= 7.0.0
  2. PDO PHP Extension
  3. Tokenizer PHP Extension
  4. OpenSSL PHP Extension
  5. XML PHP Extension
  6. Mbstring PHP Extension
  7. GD Library
  8. CURL Library
  9. PHP mod_rewrite enabled

Make sure you have a mysql database already created, and also make sure that the database schema have already been imported.

B) Installation Process - top

First, let us say that in your hosting server, you have current /public_html/ directory, which is accessible publicly via web domain, for example:

/home/hosting-user/public_html/

Now, create a new directory, ( let's name the directory "script" ) inside the /public_html/, for example:

/home/hosting-user/public_html/script/

Now open the /script/ directory in your server and upload the application.zip file downloaded from your purshase download page and unzip it.

At this point, you can find a ".env" file in the /public_html/script/ directory, if not, make sure the option to see hidden files is enabled on your hosting server.

Open the "public_html/script/.env" file and edit the database information , APP_NAME, APP_URL, and email information.

APP_NAME= phpWallet
APP_ENV=local
APP_KEY=base64:NY0K0NvBZJKYObAyHI9v90CyQBKFxA8+1rWqETYoAEQ=
APP_DEBUG=true
APP_LOG_LEVEL=debug
APP_URL=http://your-domain.com

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=Your database name
DB_USERNAME=Your database user
DB_PASSWORD=Your database password

BROADCAST_DRIVER=log
CACHE_DRIVER=file
SESSION_DRIVER=file
QUEUE_DRIVER=sync

REDIS_HOST=127.0.0.1
REDIS_PASSWORD=null
REDIS_PORT=6379

MAIL_DRIVER=smtp
MAIL_HOST=smtp.mailtrap.io
MAIL_PORT=2525
MAIL_USERNAME=null
MAIL_PASSWORD=null
MAIL_ENCRYPTION=null

PUSHER_APP_ID=
PUSHER_APP_KEY=
PUSHER_APP_SECRET=

At this point, you can see that the site is accessible to the web, via http://yourdomain.com/script/public/ right?

Now, to make the site accessible via http://yourdomain.com , the next step is to copy all contents inside the /public_html/script/public/ directory to the top level /public_html/ directory.

Remember to copy the /script/public/.htaccess to the /public_html/ also. Don’t forget this, it is very important for your app routes. Without it, your routes won’t be working and there will be a blank or empty page in every route.

Now let’s modify the /public_html/index.php to reflect the new structure. Don’t modify the /public_html/script/public/index.php, okay? Only modify /public_html/index.php, remember this!!!

Find the following line

require __DIR__.’/../bootstrap/autoload.php’;

$app = require_once __DIR__.’/../bootstrap/app.php’;

And update them to the correct paths as following

require __DIR__.’/script/bootstrap/autoload.php’;

$app = require_once __DIR__.’/script/bootstrap/app.php’;

Almost done, it is time to make a Symbolic Link for the /script/storage/ directory, so it can reflect the new structure.

Create a new php file in /public_html/ let's name it symlink.php for example:

/home/hosting-user/public_html/symlink.php

Then add the code above, (correct the paths to reflect your server public_htm structure)


symlink( '/home/hosting-user/public_html/script/storage/app/public' , '/home/hosting-user/public_html/storage' );

Now open the recently created script on your brower by going to http://your-domain.com/symlink.php.

If every thing went well, you should see an blank page. And that means you've done the installation.

D) Admin Dashboard - top

Now that the site is up and running, register a new User to turn him into and admin.

Open your database from your cpanel phpmyadmin, select your database and go to users table.

Edit the recently created User. change the column role value from 0 to 1.

Now, back to your website go to http://your-domain.com/admin
